Dear PV Team,

Vaxzevria의 새로운 적응증 추가를 위한 규제 신청이 예정되어 있으니 미리 안내드립니다.

신청 내용:

현재 허가 적응증: COVID-19 백신 (일반 성인, 만 18세 이상)

신규 적응증: COVID-19 백신 (면역저하 환자, 장기이식 수혜자 등)

제출 국가: EU (EMA), USA (FDA), Korea (MFDS)

예상 제출 일: 2022년 1월 중

제출 유형: Type C Variation (신규 적응증)

근거 데이터:

Phase III/IV 임상 시험 결과 (Immunocompromised subgroup analysis)

시판 후 Real-world data (RWD) 수집 결과

Case report 분석 (특히 면역저하 환자 안전성/유효성 프로파일)

PV팀 협력 사항:

1. 현재까지 수집된 면역저하 환자 관련 Adverse Event 정리 (특히 백신 이상반응 사례)

2. PSUR update 예비 작성 (신규 적응증 데이터 포함)

3. 새 적응증 추가 시 RMP 업데이트 검토 (리스크 프로파일 변경 여부 확인)

4. Patient information leaflet (PIL) 업데이트 감수 (면역저하 환자용 특수 정보)

**기한:** 2022년 1월 초까지 예비 feedback 부탁드립니다.
