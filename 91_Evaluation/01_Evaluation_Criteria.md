# PSUR 보고서 생성 결과물 평가지표

## 문서 정보
- **작성일:** 2025-12-31
- **작성자:** AI 평가 시스템
- **버전:** 1.0
- **평가 대상:** LLM 기반 PSUR 보고서 자동 생성 결과물

---

## 1. 평가 목적

다양한 LLM 모델을 활용하여 생성한 PSUR(정기 안전성 갱신 보고서) 결과물의 품질을 객관적으로 비교 평가하고, 최적의 생성 방식을 도출한다.

---

## 2. 평가 대상 모델

| 모델 ID | 모델 구성 | 생성 방식 |
|---------|----------|----------|
| A | Claude Opus + Sonnet Hybrid | 2-Phase (Sonnet 초안 → Opus 개선) |
| B | Gemini 3 Pro Preview | 단일 모델 1-Pass |
| C | Gemini 3 Flash Preview | 단일 모델 1-Pass |
| D | Claude Opus | 단일 모델 (Opus only) |

---

## 3. 평가 지표 체계

### 3.1. 평가 영역 및 배점

| 영역 | 가중치 | 하위 항목 |
|------|--------|----------|
| **1. 완성도 (Completeness)** | 20% | 섹션 완성률, 필수요소 포함 |
| **2. 정확성 (Accuracy)** | 25% | 데이터 추출 정확도, 변수 치환 |
| **3. 구조 준수 (Structure)** | 15% | 템플릿 준수, 가이드라인 준수 |
| **4. 내용 품질 (Quality)** | 20% | 분석 깊이, 논리성, 전문성 |
| **5. 형식 적합성 (Format)** | 10% | 마크다운 형식, 표 구성 |
| **6. 일관성 (Consistency)** | 10% | 용어 일관성, 데이터 정합성 |
| **합계** | 100% | |

---

## 4. 세부 평가 기준

### 4.1. 완성도 (Completeness) - 20점

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 15개 섹션 완성 | 10점 | 00_표지 ~ 14_별첨 모든 섹션 포함 여부 |
| 표지 필수정보 | 2점 | 품목명, 성분명, 보고기간, 제출일, 버전 |
| 목차/표목차 | 2점 | 목차 및 표목차 완성도 |
| 약어 설명 | 2점 | 사용된 약어 정의 포함 |
| 별첨 완성 | 4점 | 첨부문서, 변경사항 등 별첨 포함 |

**채점 기준:**
- 10점: 모든 항목 완벽 충족
- 7-9점: 대부분 충족 (1-2개 미흡)
- 4-6점: 절반 이상 충족
- 1-3점: 절반 미만 충족
- 0점: 거의 미충족

---

### 4.2. 정확성 (Accuracy) - 25점

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| CS 변수 치환 | 8점 | [CS숫자] 변수의 정확한 값 치환 |
| PH 변수 치환 | 5점 | [PH숫자] 변수의 정확한 값 치환 |
| 표 변수 치환 | 5점 | [표숫자] 변수의 정확한 데이터 삽입 |
| 날짜/기간 정확성 | 4점 | 보고기간, 허가일 등 날짜 정확성 |
| 수치 데이터 정확성 | 3점 | 판매량, 이상사례 건수 등 수치 정확성 |

**채점 기준:**
- 변수 치환 정확도 = (정확히 치환된 변수 수 / 전체 변수 수) × 100%
- 95% 이상: 만점
- 80-94%: 80% 배점
- 60-79%: 60% 배점
- 40-59%: 40% 배점
- 40% 미만: 20% 배점

---

### 4.3. 구조 준수 (Structure) - 15점

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 템플릿 형식 준수 | 6점 | 제공된 15개 템플릿 형식 준수 |
| 식약처 가이드라인 | 5점 | PSUR 작성 가이드라인 준수 |
| 섹션 순서 | 2점 | 올바른 섹션 배치 순서 |
| 표 번호 체계 | 2점 | 일관된 표 번호 부여 |

**채점 기준:**
- 15점: 완벽한 구조 준수
- 12-14점: 경미한 구조 이탈 (1-2건)
- 9-11점: 일부 구조 이탈 (3-5건)
- 6-8점: 상당한 구조 이탈
- 0-5점: 구조 미준수

---

### 4.4. 내용 품질 (Quality) - 20점

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 분석 깊이 | 6점 | 안전성 평가 분석의 심층성 |
| 논리적 전개 | 5점 | 서론→본론→결론 논리 흐름 |
| 전문성 | 4점 | 약물감시 전문 용어/개념 사용 |
| 결론 도출 | 3점 | 명확한 유익성-위해성 결론 |
| 권고사항 | 2점 | 구체적이고 실행 가능한 권고 |

**채점 기준:**
- 20점: 전문가 수준의 분석
- 15-19점: 우수한 분석 품질
- 10-14점: 양호한 분석 품질
- 5-9점: 기본적 분석 수준
- 0-4점: 분석 미흡

---

### 4.5. 형식 적합성 (Format) - 10점

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 마크다운 문법 | 4점 | 올바른 마크다운 문법 사용 |
| 표 형식 | 3점 | 표의 정렬, 가독성 |
| 제목 계층 | 2점 | 일관된 제목 레벨 사용 |
| 줄바꿈/단락 | 1점 | 적절한 단락 구분 |

---

### 4.6. 일관성 (Consistency) - 10점

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 용어 일관성 | 4점 | 동일 개념에 동일 용어 사용 |
| 데이터 정합성 | 4점 | 본문/표 간 수치 일치 |
| 날짜 형식 일관성 | 2점 | 날짜 표기 형식 통일 |

## 5. 비용 효율성 평가 (별도 지표)

## 7. 평가 절차

1. **1단계:** 각 결과물별 6개 영역 개별 평가
2. **2단계:** 가중치 적용하여 총점 산출
3. **3단계:** 비용 효율성 지표 계산
4. **4단계:** 종합 등급 부여
5. **5단계:** 개선 권고사항 도출

---

## 8. 참고 사항

- 평가는 객관적 기준에 따라 수행되나, 내용 품질 등 일부 항목은 주관적 판단이 포함될 수 있음
- 모든 평가는 동일한 입력 데이터(RAW Data, 템플릿, 컨텍스트)를 기준으로 함
- 비용 효율성은 품질 점수와 별도로 산출하여 종합 판단에 활용
